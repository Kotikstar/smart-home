# Smart Home utilities

Веб-интерфейс для управления топливом, картами, пропусками и гаражом Car Book в одном приложении с общей авторизацией и поддержкой двух языков (ru/en).

## Car Book
- Открывайте `/car_book.php` чтобы вести флот, статусы машин, журнал сервисных событий и пробег.
- Все данные хранятся в той же базе `kpp`, таблицы создаются автоматически, а права доступа контролируются через пермишн `carbook`.
- Добавляйте события (ТО, бронирование, смена статуса), фиксируйте пробег и используйте быстрые кнопки смены статуса прямо из карточек.

## Переключение языка
- В верхнем меню есть селектор языка (ru/en). Выбор сохраняется в сессии и применяет переводы к навигации и Car Book.
